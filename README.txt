### How to run the Django Number App

# Inside the folder "Django", Run `python3 -m venv env`

# Then run `source env/bin/activate` to start the virtual environment
The env folder should appear inside Django/

# Install Django and Boostrap (for styling the frontend) within the env: `pip3 install django && pip3 install django-bootstrap-v5`

# Then run the Django server with `python3 manage.py runserver`

# In the terminal, `cmd + click` on Mac (or `ctrl + click` on other systems) on the address to view the page on localhost
